# BodianVIPUnlocker

波点音乐大会员解锁 & 开屏广告屏蔽插件（越狱版）

## 🧩 功能介绍

本插件适用于 iOS 越狱环境，目标应用为波点音乐（`com.wenyu.bodian`），通过动态库方式实现以下功能：

- ✅ 解锁大会员特权，无需登录、无需订阅
- ✅ 替换昵称、VIP 按钮文案、底部提示等
- ✅ 拦截开屏广告请求，显著加快启动速度
- ✅ 本地运行，无需任何代理或配置文件

## 📱 支持版本

- 波点音乐主应用：最新版（长期维护）
- 系统支持：iOS 14 ~ iOS 17（含 Rootless 越狱）
